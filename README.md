# Webview Prototype

## Installation

```bash
# brew 가 없으시다면,
$ /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

# node 가 없으시다면,
brew install node

# 소스 빌드
$ npm install -g yarn       # yarn 설치
$ yarn                      # 의존성 모듈 설치
$ yarn build                # JavaScript 빌드 실행
$ yarn start:prod           # 데모 서버 실행
```
